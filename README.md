# Glass-List

<p align="center">A task manager featuring list organization, time-based views, and a glass interface.</p>

## Features

- Task list organization with multiple lists support
- Glass-morphism design with modern aesthetics
- Intuitive task management and categorization

## Tech Stack

- TypeScript  
- React  
- Tailwind CSS

## Getting Started

### Requirements

- Node.js and npm installed

### Installation

```
npm install
npm run dev
```

Then open the URL shown in your terminal (usually `http://localhost:5173` or `http://localhost:3000`, depending on your setup).

## Usage

1. Create and organize tasks in different lists
2. Manage your tasks with an intuitive interface
3. Switch between lists to categorize your work
4. Mark tasks as complete when finished
